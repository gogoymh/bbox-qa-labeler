# CAD QA Label

이미지 또는 PDF를 업로드하고, 이미지 위에 bbox를 드래그해서 그린 뒤 설명(description)과 함께 "묶음"으로 저장하고 JSON으로 다운로드하는 React 앱입니다. 한 이미지에 여러 묶음을 추가할 수 있으며, 각 묶음은 bbox + 설명 + 워드 파일명으로 구성됩니다. (기존의 Question/Answer 입력은 제거되었습니다.)

PDF를 업로드하면 각 페이지를 브라우저에서 PNG 이미지로 변환한 뒤(페이지당 이미지 1개), 일반 이미지와 동일하게 라벨링합니다. 변환된 페이지는 `원본이름-p01.png` 형태의 이름으로 목록에 추가됩니다.

## 주요 기능

- **묶음 편집**: 도면 위에 bbox를 그리고 설명/워드를 입력한 뒤 "묶음 추가"로 저장합니다. 저장된 묶음을 클릭하면 다시 불러와 "묶음 수정"할 수 있습니다.
- **도면 조작**: 스페이스바를 누른 채 드래그하면 도면 이동(패닝), 마우스 휠로 커서 중심 확대/축소.
- **바운딩 박스**: 도면 위에서 드래그해 박스를 그리고, 박스를 클릭해 선택하면 핸들로 크기 조절·이동이 가능합니다. 각 박스에는 타이틀을 입력할 수 있고(BBox 목록의 입력란), 도면에도 표시됩니다. 선택한 박스는 `Delete`/`Backspace`/`Esc` 키로 삭제합니다.
- **패널 토글**: 왼쪽 이미지/묶음 목록과 오른쪽 묶음 편집 패널을 각각 접고 펼쳐 도면 영역을 넓게 쓸 수 있습니다.
- **자동 저장**: 작업 내용(이미지 포함)은 브라우저(IndexedDB)에 자동 저장되어, 새로고침하거나 다시 열어도 그대로 복원됩니다. JSON 다운로드 후에도 작업 내용은 초기화되지 않습니다.
- **이미지 삭제**: 업로드한 이미지를 목록에서 개별 삭제할 수 있습니다.
- **JSON 불러오기**: 내려받은 JSON을 다시 불러오면 이미지(원본 `dataUrl` 포함)와 묶음이 그대로 복원됩니다. 다른 PC/브라우저로 옮겨도 그 파일 하나로 재편집할 수 있습니다. (이미지가 없는 예전 JSON은 현재 불러온 이미지에 **파일명 기준**으로 매칭해 복원합니다.)

## 요구 사항

- Node.js 20 이상 권장
- npm

## 설치

```bash
npm install
```

## 개발 서버 실행

```bash
npm run dev
```

기본 Vite 서버가 실행되며, 터미널에 표시되는 로컬 URL로 접속하면 됩니다.

## 빌드 및 실행

아래 스크립트를 실행하면 의존성이 없을 때 `npm install`을 먼저 수행하고, 이후 프로덕션 빌드를 생성한 뒤 라벨링 앱을 `5174` 포트로 실행합니다.

```bash
./build.sh
```

브라우저에서 아래 주소로 접속해 라벨링하면 됩니다.

```text
http://127.0.0.1:5174/
```

빌드만 직접 실행하려면 npm 명령을 사용하면 됩니다.

```bash
npm run build
```

빌드 결과는 `dist/` 디렉터리에 생성됩니다.

## 다운로드 JSON 형식

JSON은 **이미지 중심 배열**입니다. 최상위가 이미지 배열이고, 각 이미지가 자신의 묶음(`records`)을 가집니다. 각 `image`에는 원본 이미지 `dataUrl`(base64)이 함께 저장되어 다른 환경에서도 그대로 다시 불러와 편집할 수 있습니다. **저장된 묶음이 있는 이미지만** 포함되며, 저장된 묶음이 하나도 없으면 다운로드되지 않습니다.

```json
[
  {
    "image": {
      "id": "image_...",
      "name": "sample.png",
      "type": "image/png",
      "size": 387527,
      "width": 1200,
      "height": 800,
      "dataUrl": "data:image/png;base64,..."
    },
    "records": [
      {
        "id": "record_...",
        "type": "description",
        "bboxs": [
          {
            "id": "bbox_...",
            "x": 0.1,
            "y": 0.2,
            "width": 0.3,
            "height": 0.25,
            "title": "밸브",
            "index": 0,
            "pixel": { "x": 120, "y": 160, "width": 360, "height": 200 }
          }
        ],
        "description": "설명",
        "wordFileName": null,
        "createdAt": "2026-07-07T00:00:00.000Z"
      }
    ]
  }
]
```

다운로드 파일명은 원본 이미지 이름을 활용해 `sample.labels.json` 형태로 생성됩니다. 여러 이미지가 섞여 있으면 첫 이미지 이름 뒤에 추가 이미지 수가 붙습니다. (기존의 `question`/`answer` 필드는 제거되었습니다.)
